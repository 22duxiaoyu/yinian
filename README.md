# Action · AI 思考工作台

Action 是一款把零散想法提炼成洞察与行动的 AI 思考工作台。产品围绕“捕捉 → 理解 → 行动 → 反馈”建立完整闭环，而不是停留在记录和搜索层面。

在线体验：https://22duxiaoyu.github.io/yinian/

## 产品能力

- 快速输入灵感、观察、复盘和行动
- 从近期输入中识别重复主题与潜在张力
- 为每条洞察展示置信度和原始证据
- 用户可确认或驳回 AI 假设，保留最终决策权
- 将 AI 建议一键转成行动，并用完成结果更新转化率
- 自动生成本周思考简报
- 全局搜索、详情查看、编辑、删除、导入和导出
- 本地账户隔离与 LocalStorage 持久化
- 桌面端、平板和移动端响应式界面

## AI 产品设计

当前版本优先验证人机协作工作流，采用本地规则引擎模拟主题聚类、证据引用和建议生成。它不会把模拟结果伪装成在线大模型能力。

第二阶段计划接入真实模型服务、云端账户和数据同步，并围绕以下指标建立评测：

- 北极星指标：洞察进入行动的周转化率
- 质量指标：洞察确认率、证据打开率、洞察驳回率
- 护栏指标：错误引用率、行动撤销率、主动建议关闭率

## 本地运行

```bash
python3 -m http.server 6002
```

打开 `http://localhost:6002`，选择“体验完整示例”即可进入预置作品集空间。

## 数据说明

当前版本的数据只保存在浏览器 LocalStorage 中。刷新或同设备回访不会丢失；换设备、换浏览器或清理浏览器数据时，需要通过 JSON 导出和导入迁移。
